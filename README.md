# 关于以SH600516与SZ000789作为投资组合的风险收益特征分析
## 一、分析目标
本次分析主要是基于马科维茨的投资组合理论，通过选取沪深的两只股票作为投资组合，来简单分析一下这个投资组合同购买单一的股票在近3年的收益与风险之间的不同，同时也确认在该投资组合下，风险最低时两只股票的持仓比例。
## 二、核心发现
首先通过导入Akshare的相关数据分析近3年方大炭素与万年青这两支股票的历史收益率与标准差，根据计算，近3年方大炭素的**收益率为-2.59%，标准差即波动率为30.23%**，万年青的**收益率为-14.23%，波动率为29.78%**，总体而言这两支股票的近3年的表现并不好。而经过将这两支进行组合并通过持仓比重的调整，可以发现该组合的收益会得到改善，波动则会减少。
**如果作为一名风险规避者，考虑最小风险的投资组合，其比例应为投资方大炭素48.49%，投资万年青51.51%，此时的收益率为-8.59%，波动率为26%，此时的夏普比率为-0.4034**。
